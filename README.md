# Colorizr README

Colorizr is a gem for easily adding color to text.

## Usage

Print a list of available colors:
```
String.colors
```
Preview the color options:
```
String.sample_colors
```
Each color is its own method and can be added to the end of a string like so:
```
require 'colorizr'
"Make this red".red
"Make this blue".blue
```
